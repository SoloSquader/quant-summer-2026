# Day 1 Proofs
## Problem 1: Prove that the sum of two even integers(x and y) is always even.

Let x and y be even integers. 
Since x is even, x can be rewritten as 2m, where m is some integer.
Since y is even, y can be rewritten as 2n, where n is some integer.

Therefore,

x + y = 2m + 2n
-> x + y = 2(m + n)

m + n is an integer because the sum of two integers is an integer. 

Therefore, x + y is equal to two multiplied by some integer. 
Therefore, x + y must be even.

## Problem 2: Prove that the sum of one even integer and one odd integer is always odd.

Let x be an even number, and y be some odd number. 

Since x is an even number, x can be rewritten as 2m, where m is any integer.
Since y is an odd number, y can be rewritten as 2n+1, where n is any integer.

Therefore,

x + y = 2m + 2n + 1
-> x + y = 2(m + n) + 1

Since m and n are integers, their sum must also be an integer. 
Since m + n are integers, two multiplied by m + n must be even.

Adding 1 to any even integer must give an odd integer. 

Therefore, 2(m + n) + 1 must be odd.

Therefore, x + y must be odd.

## Problem 3: Prove that the sum of two odd integers must be even.

Let x and y both be odd integers. 

Since x is odd, x can be rewritten as 2m + 1, where m is any integer.
Since y is odd, y can be rewritten as 2n + 1, where n is any integer.

Therefore,

x + y = 2m + 1 + 2n + 1
-> x + y = 2m + 2n + 2
-> x + y = 2(m + n + 1)

Since m, n, and 1 are all integers, their sum must also be integer. 
Multiplying any integer by two gives an even result.

Therfore,

x + y

Must be even.
