# Problem 1: Prove that if n is odd, then n² is odd.

Let n be an odd integer. n can be rewritten as 2m + 1, where m is some integer.

Therefore, 

n² = (2m + 1)^2


-> n² = 4m^2 + 4m + 1


-> n² = 2(2m^2 + 2m) + 1


Since m is an integer, 2m^2 + 2m must also be an integer. Since 2m^2 + 2m is an integer, 2(2m^2 + 2m) must be even. 


Since 2(2m^2 + 2m) must be even, 2(2m^2 + 2m) + 1 must be odd. 


Therefore, is n is odd, n² must also be odd. 


# Problem 2: Prove that if n is even, then n² is even.


Let n be an even integer. Since n is even, n can be rewritten as 2m, where m is some integer. 


Therefore, 


n^2 = (2m)^2


-> n^2 = 4m^2


-> n^2 = 2(2m^2)



Since m is an integer, 2m^2 must also be an integer. Since 2m^2 is an integer, 2(2m^2) must be even. 


Since 2(2m^2) is even, n^2 must be even. 


Therefore, if n is even, n^2 must also be even.

Since 2(2mn + n + m) must be even, 2(2mn + n + m) + 1 must be odd.
