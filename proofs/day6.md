# Problem 1: Prove that for every integer n, n² is either divisible by 4 or leaves a remainder of 1 when divided by 4.

In other words, prove that every integer square can be written as either: 4k or 4k + 1 for some integer k.


Split into cases:


Case 1: n is even.


n can be written as 2k.


n^2 = 4k^2


k is an integer. Therefore, k^2 is also an integer. Since k^2 is an integer, 4k^2 must be divisible by 4. Case proven true.



Case 2: n is odd.


n can be written as 2o + 1

n^2 = 4o^2 + 4o + 1


n^2 = 4(o^2+o) + 1


o is an integer. SInce o is an integer, o^2 + o is also integer. Since o^2 + o is an integer, 4(o^2 + o) must be divisible by 4. 


Therefore, n^2 must leave a remainer of 1 when divded by 4.  Case proven true. 


Since both cases are proven true, the orginal statement is true always.


Therefore, for every integer n, n² is either divisible by 4 or leaves a remainder of 1 when divided by 4.
