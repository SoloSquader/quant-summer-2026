# Day 4 Probability


## Work Completed

Completed an 8-problem probability set focused on combinations, permutations, binomial probability, card hands, and ordered vs unordered counting.


## Results

Problem 1:

Question: A group has 12 people. Count the number of different groups of 4.

Answer: 495

Result: Correct.


Problem 2:

Question: A group has 12 people. Count the ways to fill President, Vice President, Secretary, and Treasurer.

Answer: 11880

Result: Correct.


Problem 3:

Question: A fair coin is flipped 8 times. Find the probability of exactly 5 heads.

Answer: 21.875%

Result: Correct.


Problem 4:

Question: A fair die is rolled 6 times. Find the probability of exactly two 6s.

Answer: 20.0938%

Result: Correct.


Problem 5:

Question: A 5-card hand is drawn from a 52-card deck. Find the probability that the hand contains exactly 2 aces.

Initial answer: 4.6444%

Second answer: 16.26%

Result: Incorrect.


Problem 6:

Question: A bag has 6 red balls and 4 blue balls. Four balls are drawn without replacement. Find the probability that exactly 3 are red.

Answer: 38.095%

Result: Correct.


Problem 7:

Question: A 4-digit code is made using digits 0–9 with no repeated digits. Find the probability that the code starts with 7.

Answer: 10%

Result: Correct.


Problem 8:

Question: A 5-card hand is drawn. Given that the hand contains exactly 2 aces, find the probability that it contains the ace of spades.

Final answer: 50%

Result: Correct.


## Problem 5 Correction

Problem 5 was the main missed problem.

The correct setup is:

[C(4, 2) × C(48, 3)] / C(52, 5)

The mistake was trying to treat the 5-card hand like a binomial probability problem instead of counting favorable 5-card hands over total 5-card hands.
